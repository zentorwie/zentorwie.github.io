更换工作平台

Jan 23rd, 2014

寒假回到家，由于未知原因，Chrome变得非常卡，换成Chromium也一样，重装一样。

寒假准备系统学习算法，所以就不打算一直深究下去，我想暂时换一个不用怎么配置的发行版试试，所以我备份了一下/home以后就安装了LinuxMint 16,桌面环境是Cinnamon，使用起来很友好而且配置都非常直接简单。

先使用2天，如果稳定就在寒假一直用Mint。

总结一下在Mint下面搭建一个敲代码平台需要哪些东西：

  • Emacs – Emacs is my sword.
  • git – 用来从仓库里面拿.emacs和下载博客程序
  • zsh – 高效使用命令行
  • oh-my-zsh – 用zsh怎能少了它？
  • Chrome/Chromium – 浏览网页，用google帐号同步非常方便
  • octopress – 直接用git从仓库克隆到本地，这个博客的程序
  • rvm/ruby/jekyll – octopress依赖软件包
  • fcitx/fcitx-googlepinyin – 中文输入

有这些东西，一个工作平台就基本搭好了，可以愉快地敲代码了。: )
